# 📘 Student Course Tracker

## 📌 Overview
Student Course Tracker is a full-stack web application that allows users to manage and track their courses throughout a semester. Users can add, edit, delete, and monitor course progress in a simple and organized interface.

---

## 🚀 Features
- Add new courses with details (name, professor, credits, semester)
- View all courses in a structured list
- Edit and update course information
- Delete courses
- Track course status (In Progress / Completed)

---

## 🛠️ Tech Stack
- **Backend:** C#, ASP.NET Core MVC  
- **Database:** Entity Framework Core + SQLite  
- **Frontend:** Razor Views, Bootstrap  
- **Version Control:** Git & GitHub

## ⚙️ How to Run Locally

1. Clone the repository

git clone https://github.com/Ashrith100/student-course-tracker.git

cd student-course-tracker


2. Install dependencies

dotnet restore


3. Apply migrations

dotnet ef database update


4. Run the application

dotnet run


5. Open in browser

http://localhost:5192
